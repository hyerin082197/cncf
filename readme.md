Artifact Hub
설명: 클라우드 네이티브 아티팩트 관리 - Artifact Hub는 Helm 차트, OLM 번들 등 다양한 클라우드 네이티브 아티팩트를 쉽게 검색하고 관리
핵심 기능: 다양한 리소스를 중앙에서 검색 및 관리, 오픈 소스 및 상용 아티팩트 포함 (각 클러스터에서 필요한 아티팩트를 손쉽게 찾고 관리)

Backstage
설명: 소프트웨어 인프라 관리 플랫폼 - Backstage는 여러 개발 도구와 서비스를 통합하여 한 곳에서 관리
핵심 기능: 서비스 카탈로그 제공, 기술 문서 관리 및 플러그인 확장 (복잡한 개발 환경을 간소화하고 팀별 효율성을 극대화)

Buildpacks.io
설명: 소스 코드의 컨테이너 이미지 자동 빌드 - Buildpacks는 소스 코드를 자동으로 분석하여 효율적인 컨테이너 이미지를 생성
핵심 기능: 여러 언어 지원, 빌드 과정 자동화 (수동 스크립트 작성 없이 간편하게 컨테이너 이미지 생성)

Dapr
설명: 분산 애플리케이션 런타임 - Dapr은 마이크로서비스 간의 통신, 상태 저장, 비밀 관리 등의 기능을 제공하여 분산 시스템의 복잡성 감소
핵심 기능: 서비스 호출 및 상태 관리 제공, 사이드카 아키텍처 (마이크로서비스를 쉽게 개발 및 운영)

KubeVela
설명: Kubernetes 기반 애플리케이션 배포 플랫폼 - KubeVela는 선언적 배포 및 관리를 통해 Kubernetes 복잡성을 단순화
핵심 기능: OAM 기반, CI/CD 통합, 선언적 배포 지원 (애플리케이션을 Kubernetes에서 쉽게 배포 및 관리)

KubeVirt
설명: Kubernetes에서 VM 관리 - KubeVirt는 VM과 컨테이너를 동일한 인프라에서 함께 실행하고 관리
핵심 기능: VM과 컨테이너 통합 관리, CRD 기반 선언적 관리 (레거시 VM을 Kubernetes에 통합하여 관리)

Operator Framework
설명: 상태 저장 애플리케이션 관리 - Operator Framework는 복잡한 애플리케이션의 수명 주기 관리 자동화
핵심 기능: CRD와 컨트롤러를 통한 배포 및 복구 자동화 (데이터베이스 등 상태 저장 애플리케이션을 Kubernetes에서 쉽게 관리)
